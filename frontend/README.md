# SkillPilot Frontend

This folder contains the React UI for SkillPilot, the resume-to-roadmap career planning app.

## What it includes

- Resume upload and target-role selection
- Skill gap analysis and summary cards
- Roadmap generation and dependency graph visualization
- Resume strength score and recommendations
- Progress tracking for learning resources
- Floating chatbot for guidance

## Scripts

### npm start

Runs the app in development mode at http://localhost:3000.

### npm run build

Builds the production bundle into build/.

### npm test

Launches the test runner.

## Notes

- The frontend expects the Flask API to be running on http://localhost:5000.
- Keep the role list and skill lists in sync with the backend when adding new career paths.
